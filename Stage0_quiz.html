Intro

HTML is run on most if not all broswers, it is how pages are displayed in a clients browser.
A key concept is first, realizing how the internet makes this all possible.
The cycle goes a little something like this: when You request a page, for example Facebook's main page , you request for the information stored in a server to be downloaded to your local computer.
When ever you click on an image, video, or any button on the page, it is referencing another page , which is then brought back from the server to the local computer.
This is done, by using HTTP or Hypertext Transfer Protocol

HTML Marking

HTML has tags which can change the format of how it is presented. For example: to italicize a word the tag em is used , for a bold font, the tag b is used.

Attributes
There are tags, which can have attributes that link them to other pages, display video, audio, GIFs, and other pages.

Computers vs Humans

Computers today, do not have the capability that humans have to correct simple syntax errors.

Computers will take the code YOU provide, and will interpret it, line by line. If the computers does not see a closed parenthesis, or a unclosed tag,
or any other countless error, the computer can simply not guess as to what to do next .

As a side note, there are two different types of elements which we have learned thus far. They are the following:

  • Inline Element
  • Block Element

These two elemts differ, in the way the content within the tag behaves differently, for example, a block element will contain an "Invisible Box" around the content of the tag.
